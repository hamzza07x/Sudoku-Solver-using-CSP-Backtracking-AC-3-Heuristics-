# Sudoku Solver using CSP

A Python-based Sudoku solver built using Constraint Satisfaction Problem (CSP) techniques. It efficiently solves puzzles of varying difficulty using advanced search and constraint propagation methods.

## 🚀 Features

- Backtracking search
- Forward checking
- AC-3 constraint propagation
- Heuristics:
  - MRV (Minimum Remaining Values)
  - LCV (Least Constraining Value)
  - Degree heuristic
- Performance tracking:
  - Backtracking calls
  - Failures
  - Execution time

## 📊 Output

- Displays initial and solved Sudoku boards  
- Shows execution time and backtracking statistics  
- Verifies correctness of the solution  

## 🧠 Concepts Used

- Constraint Satisfaction Problems (CSP)  
- Backtracking Search  
- Constraint Propagation (AC-3)  
- Heuristic Optimization  

## 📌 Notes

- Input puzzles are automatically generated in `.txt` files  
- Empty cells are represented by `0`
